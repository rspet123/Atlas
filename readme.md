# Gauntlet
## _A 3rd party matchmaking service for Overwatch_

## Features

- Queue System
- ELO System
- In Depth Stats and leaderboards

## Tech

We Currently Use:

- [Flask] - Web Framework
- [MongoDB] - Database and User Storage
- [Discord] - Login/Auth and Discord Server Integration
